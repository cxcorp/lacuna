13       * Opens a readable {@link SeekableByteChannel} to the memory of the specified process.

15       * @param pid The ID of the process.
16       * @return a readable, non-null {@link SeekableByteChannel} to the memory of the specified process.
17       * @throws ProcessOpenException if the process does not exist, privileges are

19       *                              other reasons.
20       */
